# TaskAtm

## Pre-requisites

- .NET Core 3 Preview 8
- Git
- NodeJs v10+

## Setup

Clone the project

```
git clone git@github.com:stormwild/TaskAtm.git
```

## Build

Build the web application

```
cd TaskAtm/TaskAtm
dotnet build
```

Build the client application

```
cd ClientApp
npm run build
```

## Run

Run the application

```
cd ..
dotnet run
```
